Set up your Hippo Project

This web application makes it easier to set up a new Hippo project. You can browse the Hippo feature library, install features into your Hippo project and configure them. To get some basic understanding of Hippo's setup application and how to use it, check the documentation.

Important Notes

  • Please note that while you are using this application, it will be necessary to rebuild your project once in a while. This is caused by the fact that the plugins need to be integrated both into this web application and into your project. For the installation of certain plugins, it will be necessary to rebuild your project twice. The Rebuild indicator will show this need, and the Rebuild page tells you which plugins require a rebuild of the project.
  • Please note that if the setup application encounters a preview HST configuration in your repository, it will be deleted automatically and you will lose any pending changes. Such a preview configuration is the result of changes made in the CMS Template Composer that you forgot to publish.

Project Default Settings

The setup application will apply several settings by default. You can set the desired values below. We recommend to select add sample content to get started.
